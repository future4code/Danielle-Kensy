Descrição

SpringBot adventure foi um projeto que trabalhei com uma amiga, onde o objetivo era contar a história de um RPG e já ter acesso a ficha de criação de personagem caso quisesse jogar.

Infelizmente nunca chegamos a desenvolver o jogo o qual daria sequência nesse projeto...MAS fala sério fiz um ótimo trabalho nesse front consumindo API, e todos os personagens registrados ficam salvos nessa mesma API. Acesse o site ou API basta clicar nas Imagens abaixo :)
